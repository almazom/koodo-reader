# Шаблоны запросов на русском языке для Minimax LLM

Этот документ содержит шаблоны запросов на русском языке для функции суммаризации глав с использованием Minimax LLM.

## Основные шаблоны суммаризации

### Краткое резюме главы

```json
{
  "id": "quick_summary",
  "name": "Краткое резюме",
  "system_message": "Ты профессиональный литературный аналитик, специализирующийся на создании кратких и содержательных резюме глав книг. Твоя задача - выделить основные моменты и ключевые события главы, сохраняя при этом стиль и тон оригинального текста. Избегай спойлеров о будущих событиях. Представь информацию в сжатой, но понятной форме.",
  "user_template": "Создай краткое резюме (2-3 абзаца) следующей главы книги: \n\n{{chapter_content}}",
  "parameters": {
    "temperature": 0.7,
    "max_tokens": 600
  }
}
```

## Примечания по использованию шаблонов

1. Все шаблоны могут быть отредактированы пользователем в настройках приложения.
2. Параметр `temperature` контролирует творческую свободу модели (более высокие значения = более творческие результаты).
3. Параметр `max_tokens` ограничивает длину генерируемого ответа.
4. Плейсхолдер `{{chapter_content}}` будет заменён на текст главы при отправке запроса к API.
5. В будущих версиях будут добавлены дополнительные шаблоны для разных целей и типов книг.
